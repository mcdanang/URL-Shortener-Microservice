URL Shortener Microservice

URL Shortener

Features:

  1. You can POST a URL to [project_url]/api/shorturl and API will response with a JSON containing shortened URL.
    Example : {"original_url":"https://www.google.com","short_url":1}
  2. If URL doesn't follow the http(s)://www.example.com(/more/routes) format, the JSON response will contain an error {"error":"invalid url"}

Short URL Creation

example: POST /api/shorturl - https://www.google.com

Example Usage:

[project_url]/api/shorturl/1

Will Redirect to:

https://www.google.com
